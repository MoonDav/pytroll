This is the home of the pytroll project. The pytroll project started as a collaboration on weather satellite processing between DMI and SMHI.

Its objective is provide different python modules for the reading, interpretation, and writing of weather satellite data.

The available python packages at the moment are:

  • pyresample for resampling satellite data
  • mipp for reading weather satellite data
  • mpop for processing weather satellite data
  • python-bufr for reading bufr files
